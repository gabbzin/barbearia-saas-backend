# Regras para o seed
Todas as informações serão com dados ficticios, então pode ficar a vontade para colocar os nomes

- 3 Barbearias
- 5 barbeiros em cada barbearia
- Intervalo (barberBreak) em todos os barbeiros das 12:00 até as 14:00
- Todos os barbeiros com disponibilidade entre 08:00 até 18:00 de segunda à sabado
- 5 serviços para cada barbeiro
- 5 agendamentos para cada barbeiro em datas passadas, com o limite sendo até hoje (1 agendamento por serviço)
- 10 agendamentos aleatórios para o futuro (com data superior a atual "06/04/2026 15:40") no máximo uma semana pra frente.
- 3 usuários (1 cliente para cada barbearia) e um deles terá duas contas em duas barbearias distintas
- Algumas avaliações em cada agendamento
- 4 planos (Plano grátis, Plano Corte Ilimitado, Plano Barba Ilimitada, Plano Corte e Barba Ilimitados)
- 1 assinatura registrada e vinculada a um usuário
- Os usuários e os barbeiros devem ser criados vinculados ao BetterAuth que é o responsável por fazer a autenticação, caso haja problemas na criação me relatar e propor as alterações (falando quais as sugestões) para corrigir
